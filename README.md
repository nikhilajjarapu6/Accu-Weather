## About Accu-Weather

Accu-Weather is a sleek, responsive web application that provides accurate and real-time weather information. Built with vanilla HTML, CSS, and JavaScript, it fetches live weather data and displays it in a clean, user-friendly interface.

## Features

- Real-time weather data from API
- Beautiful, responsive user interface
- Works on desktop, tablet, and mobile devices
- Current weather conditions display
- Temperature and weather details
- Fast loading and smooth performance
- Clean and intuitive design
- Easy to navigate interface

## Tech Stack

| Component | Technology |
|-----------|------------|
| Frontend | Vanilla JavaScript |
| Styling | CSS3 |
| Markup | HTML5 |
| API | Weather API Integration |
| Deployment | GitHub Pages |

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for API calls

### Installation

1. Clone the repository
git clone https://github.com/nikhilajjarapu6/Accu-Weather.git
cd Accu-Weather

2. Open in your browser
Simply open the index.html file or use a local server:
python -m http.server 8000

Then visit: http://localhost:8000

### Live Demo
Accu-Weather is deployed on GitHub Pages:
https://nikhilajjarapu6.github.io/Accu-Weather/

## How to Use

1. Open the application in your browser
2. Allow location access for automatic weather detection
3. OR enter a city name to search for weather
4. View current weather conditions
5. Check temperature, humidity, and other details
6. See day/night themed weather icons
